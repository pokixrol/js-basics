2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. JavaScript is case sensitive
  2. Commands must end with a semicolon if there are more than one on one line (ending with semicolon is also suitable for individual orders)
  3. The command block is preceded by curly braces - {...}
  4. One-line comment is marked with //text
  5. multiline comment is marked with /*text*/

Doplňte další možnosti ve formě kódu.

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Na stránce se objeví vyskakovací okno s textem "Nezadal jsi správné heslo!" */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
var password = prompt("Zadejte heslo:", "1234");

confirm()

/*Uživatel rozhodne zda-li chce přejít na stránku seznam.cz pomocí talčítek "ok" nebo "zrušit"*/
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:
1.Vyskakovací okna často blokuje adblock
2. Můžo obsahovat nehťený obsah
